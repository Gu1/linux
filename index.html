Goal of the project

As you probably know if you own an ODROID device by the Korean company hardkernel, the kernel they officially support on their devices is rather old (3.0.y) is also oriented for a desktop usage..

For the these reasons, I've decided to maintain my own kernel for the ODROID (U2 only for now) with the goal of being as close as possible to mainline but without support for fancy features like 3D acceleration, HDMI, etc... Therefore, my kernel is clearly oriented for server use.

This project was started because I needed to be able to run a grsecurity-enabled kernel on the ODROID-U2, so you should be able to apply the patch on top of my kernels.
